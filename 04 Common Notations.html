Maths notations

This page details the maths notations that you should be familiar with.

Number classes

Notation Description
$\Bbb{W}$ The set of whole numbers {0,1,2,3,4,5,...}, we can think of these as unsigned integers.
$\Bbb{N}$ The set of whole numbers above zero {1,2,3,4,5,...}, we can think of these as "true" unsigned integers. That is if a variable was 0 it would evaluate to "false", so any positive value evaluate to "true".
$\Bbb{Z}$ The set of positive and negative whole numbers {...-2,-1,0,1,2,3,4,5,...}, we can think of these as signed integers.
$\Bbb{Q}$ The set of rational numbers, this means that the value can be expressed as a fraction. We can represent these as floating point values, but need to be aware of precision and accuracy issues.
$\Re$ or $\Bbb{R}$ The set of real numbers, we can think of these as floating point values, but need to be aware of precision and accuracy issues.
$\Im$ or $\Bbb{C}$ The set of imaginary or complex numbers. These are represented as Complex number, a pair of floating point numbers.
struct Complex { float real; float imag; }

Variables

Notation Description
$a,b,c,u,v,w,x,y,z$ magnitude variables
$\psi, \phi, \theta$ angle variables
$z$ common complex variables

Operations

Notation Description
$+,-,/,*,mod$ Standard arithmetic operators
$x^y$ x to the power of y e.g. $2^3 = 2*2*2, x=2; y=3$
$\backslash$ Takes the integer result of a division e.g. 1/3 is 0, 5/2 is 2
$\sum_{i=1}^n{x_i}$ Add a discrete list of numbers $x$ indexed by 1 and some integer n - think of this as a for loop
$\prod_{i=1}^n{x_i}$ Multiply a list of numbers $x$ indexed by 1 and some integer n - think of this as a for loop
$\int$ Integration, you can think of this as adding all numbers in a continuous range
$\frac{dy}{dx}$ Differentiation, you can think of this as a kind of subtraction

Functions

Notation Description
$\lfloor x \rfloor$ The floor function rounds a value down to the nearest integer. e.g $\lfloor 2.3456 \rfloor$ is $2$, $\lfloor 2.9999 \rfloor$ is $2$
$\lceil x \rceil$ The ceiling function rounds a value up to the nearest integer. e.g $\lceil 2.3456 \rceil$ is $3$, $\lceil 2.9999 \rceil$ is $3$

Other

  • $i$ represents the imaginary part of a complex number
  • $()[]\{\}$ group sets of operators and variables together - just like in a programming language
